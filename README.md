# ETL Marketplace

This repository contains the source code for the ETL Marketplace project, organized as a monorepo.

## Modules

### [Email](./packages/email)

A React-based email development environment.

- **Purpose**: Designing, developing, and exporting email templates using React Email.
- **Tech Stack**: React Email, Tailwind CSS.

### [ETL](./packages/etl)

Core ETL and automation services.

- **Purpose**: Handles data extraction, transformation, and loading tasks. Includes scripts for automation and database management.
- **Tech Stack**: TypeScript, Prisma (MySQL), Puppeteer, Liferay Headless Client, Ky.
